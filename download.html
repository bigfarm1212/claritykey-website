Common Questions

Does it work in Word?

Yes. ClarityKey works in Microsoft Word, Google Docs, Outlook, and any other application where you can select and copy text.

Is it free?

The core features of ClarityKey are completely free to use during our public beta. We focus on accessibility first.

Can I pause it?

Absolutely. Right-click the ClarityKey icon in your system tray to pause correction or change modes at any time.
